About

Graphy.js is a free and open source Javascript library to display graphical functions for the Web, powered by HTML5 Canvas, Javascript, Mathquill, Jasmine, npm and Gulp. The project is initiated, built and maintained by @timnqp.

The library is made to help edtech products in display graphical functions for secondary and high school education.
